## How to contribute to Flow.Launcher.Plugin.RemoteDesktop

### Bugs

Found a bug?
Search the issues if it was already reported, otherwise open an issue using the Bug template and describe how to reproduce.
Include the version of Flow.Launcher and the Plugin.

### Features

Missing a functionality?
Search the issues if it was already requested, otherwise open an issue using the Feature template and describe what you want to achieve and if there's a current workaround.

### Pull Requests

Before making a pull request, open an Issue and link it in the PR.

To create an PR, first create a fork and a new branch for the issue.
The branch must include the issue number.
The PR must target the default branch.

Commits must use [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/).
